# COVID-19 Single-Cell RNA-seq Analysis (GSE149689)

## Overview
Single-cell RNA sequencing (scRNA-seq) analysis of bronchoalveolar lavage fluid (BALF) samples from COVID-19 patients.

Groups: Severe COVID-19, Mild/Asymptomatic, Influenza, Healthy

## Workflow
1. Data loading and preprocessing  
2. Quality control (QC filtering)  
3. Normalization and scaling  
4. PCA (dimensionality reduction)  
5. Clustering (cell populations)  
6. UMAP visualization  
7. Differential gene expression analysis

## Tools
- R  
- Seurat  
- dplyr  
- ggplot2

## Notes
Raw data files are not included. Data can be downloaded from GEO: GSE149689
